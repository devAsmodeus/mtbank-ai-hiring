# Политика безопасности

## Сообщить об уязвимости

Откройте приватный security advisory через вкладку **Security → Report a
vulnerability** в репозитории на GitHub. Пожалуйста, не публикуйте детали в
открытых issue до выпуска исправления.

## Замечания по демо-развёртыванию

Это прототип. Перед публичным развёртыванием:

- смените дефолтные ключи (`PIPELINES_API_KEY`, пароль Grafana);
- порт Pipelines (9099) и Prometheus (9090) не публикуйте наружу (в
  `docker-compose.yml` они привязаны к `127.0.0.1`);
- закройте Analysis Engine (`8000`) за reverse-proxy с авторизацией - REST
  раскрывает содержимое звонков через `/trends`;
- не храните реальные записи звонков (персональные данные) в открытом хранилище.
